# fitreader-rs
Reading / writing Fit files in Rust. (Prototype)

# Logger
It uses env_logger: https://docs.rs/env_logger/latest/env_logger/

Uses RUST_LOG to set the logging level.

```shell
RUST_LOG=debug exec
```
